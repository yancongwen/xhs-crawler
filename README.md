## 小红书数据爬虫

爬取小红书某一个标签下的数据

### 参数说明

- page_id：话题分享页的id，如 5f40db4e2e11630001b3a440
  - App 点击标签，进入标签列表页，分享复制连接即可获得
- file_name：保存名称文件
- min_size：需要爬取文章的上限，爬取成功超过该值后中止程序
- sort_by：排序，可取值为 hot 与 time，默认为hot，表示按热度排序，time表示按最新排序
